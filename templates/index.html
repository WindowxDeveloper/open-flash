Open Flash

Open Flash is a Free and Open Source project to preserve SWF files passed the end of support of Adobe's Flash Plugin through the creation of a player for the web web and desktop. All the components are written in Rust and Typescript and are available small, reliable and reusable libraries to enable easy integration with your projects.

Check the player demo.

What is Open Flash

Player

The main goal of the project is to create a player for SWF files. See domu-player.

Libraries

All the components are provided a small reusable and reliable libraries.

Database

Open Flash maintains a registry of public SWF files and a database of test samples.

Documentation

Open Flash centralizes documentation around the SWF file format.
